# Zephyrus

Zephyrus (Zeph) is a prototype crypto intelligence web app built to simulate the full suite of skills for the West Wind of Crypto Twitter.

## Features

- Alpha Carrier: real-time sentiment analysis of Crypto Twitter.
- Wind flow chart analysis: momentum, trend direction, liquidity flows.
- Alpha Distribution: narrative rotations, sector rotations, hype cycles.
- Wallet and smart contract flow analysis.
- Real-time token sniping alerts.
- DeFi yield optimization suggestions.
- NFT trend forecasting.
- High-quality tweet generation (4–8 tweets/day).

## Run locally

```bash
npm install
npm run dev
```

Open the local development server URL shown after startup.
